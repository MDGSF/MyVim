# vscode

- 光标跳转到行首: `HOME`
- 光标跳转到行末: `End`
- 在当前光标行下面插入新的空行: `Ctrl + Enter`
- 在当前光标行上面插入新的空行: `Ctrl + Shift + Enter`
- 光标按单词跳动: `Ctrl + Left/Right`
- 光标选中单词: `Ctrl + Shift + Left/Right/Up/Down`
- 打开/关闭终端: `Ctrl + ~`, 就是 Esc 下面那个键
- 多行光标: `Alt + Shift + Mouse Left`
- 光标在匹配的两个括号之间跳转: `Ctrl + Shift + \`

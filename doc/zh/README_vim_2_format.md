# Vim 格式化

`==` 对当前行排版，一般来说是根据上一行来对齐。

`==` 也可以对 VISUAL 模式下选中的文本进行排版。

`gg=G` 对整个文档进行排版。

`<N>==` 对当前行开始的 N 行进行排版。

`=<N>j` 从当前行开始向下 N 行进行排版。

`=<N>k` 从当前行开始向上 N 行进行排版。

`>>` 向右缩进当前行。

`<<` 向左缩进当前行。

`2>>` 向右缩进两行。

`>G` 从当前行到文档末尾向右缩进。

`<G` 从当前行到文档末尾向左缩进。

`>gg` 从当前行到文档开头向右缩进。

`<gg` 从当前行到文档开头向左缩进。

# vimtutor

## vim 30 分鐘入門指導

在 `shell` 下直接輸入 `vimtutor`，然後就會進入一個 vim 自帶的學習指導。

跟著操作一遍，基本上就可以使用 vim 了。

## 中文版 vimtutor

如果你不願意閱讀英文，那我為你準備了一份中文版的 `vimtutor_ch.txt`。

你可以到目錄 `vimtutor` 下執行 `./tutor_zh_cn.sh` 命令來啟動，這會複製一個副本。

你也可以直接打開 `tutor.zh_cn.utf-8` 文件來學習，不過最好先複製一個副本，
因為在學習的過程中會修改文檔。

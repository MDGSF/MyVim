# Text object selection

```
:help motion.txt

6. Text object selection
```

- `aw`: "a word"
- `iw`: "inner word"
- `aW`: "a WORD"
- `iW`: "inner WORD"
- `as`: "a sentence"
- `is`: "inner sentence"
- `ap`: "a paragraph"
- `ip`: "inner paragraph"
- `a]`, `a[`: "a [] block"
- `i]`, `i[`: "inner [] block"
- `a)`, `a(`, `ab`: "a block"
- `i)`, `i(`, `ib`: "inner block"
- `a>`, `a<`: "a <> block"
- `i>`, `i<`: "inner <> block"
- `at`: "a tag block"
- `it`: "inner tag block"
- `a}`, `a{`, `aB`: "a Block"
- `i}`, `i{`, `iB`: "inner Block"
- `a"`, `a'`, `` a` ``: "a quoted string"
- `i"`, `i'`, `` i` ``: "a quoted inner string"

### a word

```
aw
```

# Vim 文件管理器

```
:help :Ex
```

## 打開文件管理器

```
:Ex          在當前窗口打開
:Explore     在當前窗口打開
:Hexplore    在水平窗口打開
:Lexplore    在左側窗口打開
:Sexplore    在水平窗口打開
:Vexplore    在左側窗口打開
:Texplore    在新的標簽頁打開
```

如果你執行了 `:Ex` 在當前窗口打開了文件管理器，又想回到之前的文件，
可以使用 `:bdel` 當前窗口的緩衝區刪掉。

# vim 基础配置介绍

vim 默认配置放在 `/etc/vim/vimrc`，个人用户配置文件放在 `~/.vimrc`，
正常情况下我们只修改个人的配置文件 `~/.vimrc`，如果自己的 Home 目录下
没有这个文件，就新建一个 `touch .vimrc`。

下面介绍的内容都在 `~/.MyVim/vimrcs/basic.vim`（可以用 `gf`
跳转过去先看看，`ctrl+^` 可以跳转回来）。

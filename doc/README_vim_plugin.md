# vim 常用插件

## [Nerd Tree](https://github.com/scrooloose/nerdtree)

打开目录树。

```
<leader>nn
```

## ack.vim

在当前目录和子目录下搜索字符串

```
<leader>g
```

## bufexplore

打开 bufexplore

```
map <leader>o :BufExplorer<cr>
```

## MRU.vim

打开 MRU 面板，查看最近使用的文件。

```
map <leader>f :MRU<CR>
```

## ctrlp.vim

搜索文件

```
let g:ctrlp_map = '<c-f>'
map <leader>j :CtrlP<cr>
```
